종방향 속도 제어 — 일정 간격(Constant-Space) 추종

Chapter 3 · Vehicle Control / 02_constant_space  |  앞 차량과 고정 거리를 유지하는 종방향 PID

앞서 달리는 차량을 일정한 거리(기본 20 m)를 두고 따라가는 종방향 PID 제어기를 설계해 보세요. 두 대의 차량이 등장하는 첫 번째 추종 시나리오입니다.

1문제 상황

앞서 달리는 target 차량과 뒤따르는 ego 차량이 있습니다. ego 는 target 과의 거리(target_x − ego_x)를 목표 간격 target_space = 20 m으로 유지해야 합니다.

  • target 은 일정 속도(vx = 10 m/s)로 주행하며, ego 는 PID 로 간격을 조절합니다.
  • ego 에는 drag 가 작용하므로, PD-only 로는 작은 잔류 오차가 남습니다 — 이것이 이번 과제의 학습 포인트입니다.
  • 충돌(gap < 0) 없이 간격이 발산하지 않으면 합격입니다.

2플랜트 모델

ego·target 모두 vehicle_long_space.py의 VehicleLong 종방향 점질량 모델입니다.

  • target: drag 없음(Ca=0), 가속도 명령 = 0 (등속).
  • ego: drag 포함(Ca=0.5), 가속도 명령 = 여러분 제어기의 출력 u.
  • 초기 조건: target.x0=30, ego.x0=0 → 초기 간격 30 m (목표 20 m 보다 멀리 시작).
  • 두 차량 모두 초기 속도 vx0=10 m/s.

3구현 과제

constant_space_pid.py의 ConstantSpacePID.step() 메소드를 구현하세요. 

class ConstantSpacePID:
    def __init__(self, kp: float, kd: float, ki: float, dt: float,
                 target_space: float = 20.0): ...

    def step(self, target_x: float, ego_x: float) -> float
        """한 스텝의 가속도 명령 u 를 반환."""
인자의미
target_xtarget 차량의 현재 위치 [m]
ego_xego 차량의 현재 위치 [m]
반환가속도 명령 u — ego 의 VehicleLong.step() 입력
  • 오차는 컨트롤러 내부에서 계산합니다: error = (target_x − ego_x) − target_space.
  • 첫 호출은 D 항 = 0.

게인·파라미터 튜닝

이 과제에서 구현하는 것은 알고리즘 로직 입니다 — 라이브러리 코드의 시그니처는 고정이고 본문(# TODO)만 채웁니다. 게인·파라미터의 실제 값 은 코드가 아니라 실행 파일에서 정하며, 세 군데의 역할이 다릅니다.

  • 라이브러리 코드 (구현 대상) — 알고리즘만 구현. 게인·파라미터는 매개변수로 받기만 하고 값은 담지 않음.
  • producer (record_gen.py / demo.py) — 게인·파라미터 값을 정하는 곳. release 기본값은 0 이라 그대로 실행하면 응답이 없거나 오류가 납니다 → 직접 값을 채워 실행하며 응답 변화를 비교하세요 (자유롭게 변경 OK).
  • test_*.py — 합격 기준 검증용 값. 변경 금지.

5합격 기준 pytest

알고리즘 형태는 자유 — 동작 명세(behavioral spec)만 검증합니다.

  1. 충돌 없음 — 80초 시뮬 동안 min gap > 5 m.
  2. 정상상태 수렴 — tail 평균 |gap − target_space| < 3 m (drag 로 인한 소폭 offset 허용).
  3. 발산 없음 — peak |gap − target_space| < 12 m.
학습 포인트: PD-only 로는 drag 외란이 잔류 오차를 남깁니다. 이 오차를 직접 경험하는 것이 03_time_gap 의 time-gap 정책이 왜 유리한지를 이해하는 열쇠입니다.

6힌트

  • 오차 부호를 확인하세요: (target_x − ego_x) − space. 멀면 양수 → 가속, 가까우면 음수 → 감속.
  • D 항은 오차의 변화율 — 상대 속도를 부드럽게 반영해 충돌 직전 제동을 도와줍니다.
  • I 항(ki)을 넣으면 잔류 오차를 줄일 수 있지만, 이번 합격 기준은 PD-only 로도 통과 가능합니다.
