좌표 변환 + 다항식 피팅 — Frame Transform

Chapter 3 · Vehicle Control / 05_frame_transform  |  곡선 경로 제어(06/07/08/09)의 핵심 수학 유틸리티

글로벌 좌표계의 경로 점들을 ego 의 로컬 좌표계로 변환하고, 그 점들에 다항식을 피팅한 뒤, 원하는 x 위치의 y 값을 평가하는 세 가지 유틸리티 클래스를 구현해 보세요. 이 모듈은 06/07/08/09 에서 공통으로 사용됩니다.

1문제 상황

곡선 경로를 따라가는 제어기는 "내 앞에 경로가 어떻게 생겼는가?"를 알아야 합니다. 글로벌 좌표의 경로 점들을 ego 기준 로컬 프레임으로 바꾸고, 이를 다항식으로 근사하면 어느 제어기든 통일된 형식으로 경로 정보를 사용할 수 있습니다.

  • Global → Local 변환: 회전 + 평행이동으로 ego 좌표계 기준 점 좌표를 계산.
  • 다항식 피팅: 변환된 점들에 최소제곱(LSQ) 으로 degree-3 다항식 계수를 추정.
  • 다항식 평가: 주어진 x 배열에서 다항식 y 값을 계산 (lookahead, viz 용).

2주어진 환경

이 모듈은 시간 시뮬레이션이 없는 순수 수학 유틸리티입니다. 플랜트 모델 대신 다음 테스트 픽스처가 주어집니다:

  • 알려진 수학적 입력 (좌표 점, 다항식 계수)으로 출력을 수치 정확도로 검증.
  • 이 모듈의 출력(계수 배열)은 06/07/08/09 의 제어기가 그대로 입력으로 사용합니다.

3구현 과제

frame_transform.py에서 세 클래스를 모두 구현하세요.

① Global2Local

class Global2Local:
    def __init__(self, num_points: int): ...
    def convert(self, points: np.ndarray, yaw_ego: float,
                x_ego: float, y_ego: float) -> np.ndarray
        # (num_points, 2) local frame points — self.local_points 에도 저장
인자의미
points글로벌 좌표 점 배열 (N, 2)
yaw_egoego 의 현재 heading 각도 [rad]
x_ego, y_egoego 의 글로벌 위치 [m]
반환로컬 좌표 점 배열 (N, 2), self.local_points 에도 저장

② PolynomialFitting

class PolynomialFitting:
    def __init__(self, degree: int, num_points: int): ...
    def fit(self, points: np.ndarray) -> np.ndarray
        # (degree+1, 1) column — self.coeff 에도 저장
인자의미
points로컬 좌표 점 배열 (N, 2)
반환다항식 계수 배열 (degree+1, 1), 고차 → 저차 순서, self.coeff 에도 저장

③ PolynomialValue

class PolynomialValue:
    def __init__(self, degree: int, num_x: int): ...
    def calculate(self, coeff: np.ndarray, x: np.ndarray) -> np.ndarray
        # (num_x, 1) y values — self.y, self.points 에도 저장
인자의미
coeff다항식 계수 (degree+1, 1), 고차 → 저차
x평가할 x 값 배열
반환y 값 배열 (num_x, 1), self.y/self.points 에도 저장
계수 순서 주의: coeff[0]이 최고차 항(x^degree)의 계수, coeff[-1]이 상수항입니다. 06/07/08/09 가 이 순서에 의존합니다.

게인·파라미터 튜닝

이 과제에서 구현하는 것은 알고리즘 로직 입니다 — 라이브러리 코드의 시그니처는 고정이고 본문(# TODO)만 채웁니다. 게인·파라미터의 실제 값 은 코드가 아니라 실행 파일에서 정하며, 세 군데의 역할이 다릅니다.

  • 라이브러리 코드 (구현 대상) — 알고리즘만 구현. 게인·파라미터는 매개변수로 받기만 하고 값은 담지 않음.

  • test_*.py — 합격 기준 검증용 값. 변경 금지.

이 문제는 따로 튜닝할 게인·파라미터가 없습니다 — 알고리즘 구현에만 집중하세요.

5합격 기준 pytest

동작 시뮬이 아닌 수치 정확성으로 검증합니다.

  1. 회전 + 평행이동 — yaw=π/2, ego at (10, 5)에서 global (11, 5) → local (0, −1) (tolerance 1e-12).
  2. 다항식 피팅 정확도 — 알려진 cubic 의 sample 점들로 fit → 계수가 정답과 일치 (tolerance 1e-9).
  3. 다항식 평가 정확도 — coeff = [0, 0, 2, -1] → y = 2x − 1이 모든 x 에서 일치.

6힌트

  • 회전각: θ = −yaw_ego (음수 부호). ego 좌표계로 돌리려면 반대 방향으로 회전해야 합니다.
  • LSQ 정상방정식: coeff = (AᵀA)⁻¹Aᵀb — np.linalg.inv를 사용하세요.
  • 설계 행렬 A의 열 순서에 주의: A[i][j] = x_i^(degree−j).
