Kalman Filter 2D — 위치와 속도를 동시에 추정하기

Chapter 1 · Filters / 05_kalman_filter_2d  |  행렬 상태공간으로 확장된 칼만 필터 (위치만 측정, 위치+속도 추정)

위치 센서 하나만 있어도 위치와 속도를 동시에 추정할 수 있습니다. 04 의 scalar 칼만을 2차원 상태 벡터 + 행렬 산술로 확장해 보세요.

1문제 상황

자율주행·로봇 분야에서 가장 흔한 패턴은 "위치는 측정할 수 있지만 속도는 직접 측정이 어렵다" 는 것입니다. 칼만 필터는 위치(pos)와 속도(vel)를 한 상태 벡터로 묶고, 위치 측정만으로 두 값을 동시에 추정합니다.

이 모듈의 KalmanFilter2D 는 시스템 행렬(A, B, C, Q, R)만 바꾸면 등속 운동(CV)과 스프링-매스-댐퍼(SMD) 두 시스템을 모두 처리할 수 있습니다. 알고리즘 구조는 04 와 완전히 동일하고, 스칼라 연산이 행렬 연산으로 대응됩니다.

2주어진 환경

__init__ 은 이미 작성되어 있습니다. 시스템 행렬 A(2×2), B(2,), C(2,), 공분산 Q(2×2), R(scalar), 초기 상태 x0, P0 가 그대로 저장됩니다.

데모가 두 개입니다.

  • cv_demo.py — 등속 운동(Constant Velocity): 위치만 측정해 위치+속도 추정, 2개 서브플롯.
  • smd_demo.py — 스프링-매스-댐퍼 자유 진동: 같은 클래스에 다른 행렬 셋업.

두 데모의 Q, R 기본값은 0 — 직접 채워서 실행하세요. test_*.py 의 값은 변경 금지입니다.

3구현 과제

kalman_filter_2d.py 의 KalmanFilter2D.step() 메소드를 구현하세요. 04 의 6줄 알고리즘을 행렬 형태로 옮기는 것이 핵심입니다. 

class KalmanFilter2D:
    def __init__(self, A: np.ndarray, B: np.ndarray, C: np.ndarray,
                 Q: np.ndarray, R: float,
                 x0: np.ndarray | None = None,
                 P0: np.ndarray | None = None): ...

    def step(self, measurement: float, control_input: float) -> np.ndarray
        """한 번의 Predict + Update 를 수행하고 갱신된 상태 벡터(길이 2)를 반환."""
인자/반환의미
A 상태 전이 행렬 (2×2 ndarray)
B 제어 입력 행렬 (길이 2 1D ndarray)
C 측정 행렬 (길이 2 1D ndarray) — 위치만 측정 시 [1, 0]
Q 프로세스 노이즈 공분산 (2×2 ndarray)
R 측정 노이즈 분산 (scalar)
반환 길이 2 numpy 배열 — state[0] = 위치, state[1] = 속도
  • 반환은 1D 배열 (2,) — column vector (2, 1) 형태로 짜면 인터페이스가 깨집니다.

게인·파라미터 튜닝

이 과제에서 구현하는 것은 알고리즘 로직 입니다 — 라이브러리 코드의 시그니처는 고정이고 본문(# TODO)만 채웁니다. 게인·파라미터의 실제 값 은 코드가 아니라 실행 파일에서 정하며, 세 군데의 역할이 다릅니다.

  • 라이브러리 코드 (구현 대상) — 알고리즘만 구현. 게인·파라미터는 매개변수로 받기만 하고 값은 담지 않음.
  • producer (record_gen.py / demo.py) — 게인·파라미터 값을 정하는 곳. release 기본값은 0 이라 그대로 실행하면 응답이 없거나 오류가 납니다 → 직접 값을 채워 실행하며 응답 변화를 비교하세요 (자유롭게 변경 OK).
  • test_*.py — 합격 기준 검증용 값. 변경 금지.

4검증 시나리오

① 등속 운동 (CV) cv_demo.py

truth: pos = v·t (v=2.0 m/s 등속). 위치만 N(0, 0.5) 노이즈 측정. 300 스텝 동안 위치·속도를 동시에 추정 — warm-up(100 스텝) 이후 두 RMS 기준을 통과해야 합니다.

② 스프링-매스-댐퍼 (SMD) smd_demo.py

m=10, k=100, b=2 자유 진동. 위치만 노이즈 측정. 같은 KalmanFilter2D 에 다른 행렬 셋업만 바꿔 사용합니다. 이 시나리오는 시각적 확인용 — pytest 합격 기준은 CV 기준만 적용됩니다.

7힌트

  • 04 의 6줄 알고리즘을 행렬 형태로 옮기는 것이 핵심입니다 — TODO 주석에 수식이 그대로 제시되어 있습니다.
  • 1D 배열 (2,) 사용: np.array([0.0, 0.0]) — column vector (2, 1) 아닙니다.
  • np.outer(K, C) 가 (2,2) 행렬입니다 — K @ C(inner product, scalar)와 혼동하지 마세요.
  • A.T(전치) 빼먹기 쉽습니다 — P_pred = A @ P @ A.T + Q.
  • B * u(element-wise broadcast) 와 B @ u(내적, 차원 mismatch 가능) 를 구분하세요.
