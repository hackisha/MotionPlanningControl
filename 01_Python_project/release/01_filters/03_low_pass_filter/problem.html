Low-Pass Filter — 이전 출력과 현재 입력의 가중 평균

Chapter 1 · Filters / 03_low_pass_filter  |  단 하나의 파라미터 α 로 평활 강도를 조절하는 1차 IIR 필터

buffer 없이 이전 출력값과 현재 측정값의 가중 평균만으로 신호를 부드럽게 만드는 저역 필터(Low-Pass Filter) 를 설계해 보세요.

1문제 상황

Moving Average 는 최근 N개를 저장하는 buffer 가 필요합니다. Low-Pass Filter(LPF) 는 이전 출력 하나만 기억하면 됩니다. 파라미터 α(알파) 하나로 얼마나 과거를 "믿을지"를 조절합니다.

  • α 가 크면(1에 가까울수록) 출력이 천천히 변합니다 — 부드럽고 느린 반응.
  • α 가 작으면(0에 가까울수록) 입력을 그대로 통과시킵니다 — 거칠고 빠른 반응.

데모에서는 sin(t) 신호에 잡음을 섞은 뒤 LPF 로 평활화합니다. α 를 바꾸면서 반응 속도와 평활 강도의 트레이드오프를 확인해 보세요.

2주어진 환경

__init__ 은 이미 작성되어 있습니다. self.alpha 와 self.y = None 이 초기화됩니다. self.y 가 None 인 시점이 첫 호출임을 나타냅니다.

demo.py 는 sin(t) + N(0, 0.5) 신호 400 표본을 LPF 로 평활화합니다. demo.py 의 alpha 기본값은 0.0 (즉시 통과) 입니다 — 직접 채워 응답 변화를 비교하세요.

3구현 과제

low_pass_filter.py 의 LowPassFilter.step() 메소드를 구현하세요. 

class LowPassFilter:
    def __init__(self, alpha: float = 0.9): ...

    def step(self, x: float) -> float
        """새 측정값 x 를 받아 평활된 출력을 반환."""
인자/반환의미
x 새로 들어온 측정값
alpha 평활 계수 ∈ [0, 1] — 클수록 부드럽고 느림, 작을수록 거칠고 빠름
반환 평활된 출력값 (첫 호출에서는 x 그대로)
  • 첫 호출에서는 self.y 가 None — 이전 출력이 없으므로 x 자체를 반환하고 self.y 를 초기화합니다.
  • 이후 호출에서는 이전 출력과 현재 입력을 α 로 가중 평균합니다.

4실행 방법

모든 명령은 git 저장소 루트에서 실행합니다.

구현 파일

01_Python_project/release/01_filters/03_low_pass_filter/low_pass_filter.py

테스트 — 합격 기준 검증

uv run pytest 01_Python_project/release/01_filters/03_low_pass_filter/ -v

데모 — 시각적 확인 (구현 후)

uv run python 01_Python_project/release/01_filters/03_low_pass_filter/demo.py

demo.py 의 alpha 기본값은 0.0 입니다. 원하는 값(예: 0.9)으로 바꿔 실행하며 평활 강도를 조절해 보세요. test_*.py 의 값은 변경 금지입니다.

5합격 기준 pytest

알고리즘 형태는 자유 — 동작 명세만 검증합니다.

  1. 상수 입력 안정성 — alpha=0.9, step(2.0) 을 200회 반복 후 출력이 2.0 근방 (오차 < 1e-3).
  2. 노이즈 추적 RMS — alpha=0.9, 상수 truth=5 + N(0, 1) 노이즈 1만 표본, warm-up 이후 RMS 오차 < 0.3.

6힌트

  • 이전 출력과 현재 입력의 가중 평균 형태: y = α · y_prev + (1 - α) · x
  • α 와 (1-α) 위치에 주의하세요 — 부호/순서가 틀리면 측정 무게가 잘못 적용됩니다.
  • self.y is None 인 첫 호출 분기가 필요합니다.
  • α 직관: 크면 부드럽고 느림, 작으면 거칠고 빠름.

7제약

  • test_low_pass_filter.py — 수정 금지 (검증 환경).
  • step() 의 시그니처 변경 금지.
  • demo.py 의 alpha 값은 자유롭게 변경 가능 — test_*.py 의 값은 변경 금지.
